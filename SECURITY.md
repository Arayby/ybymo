# Política de Segurança

## Versões Suportadas

As correções de segurança são aplicadas apenas às versões atualmente suportadas.

| Versão | Suporte |
| ------ | ------- |
| 1.1.x  | :white_check_mark: |
| 1.0.x  | :white_check_mark: |
| < 1.0  | :x: |

## Reportando uma Vulnerabilidade

Se você identificar uma vulnerabilidade de segurança, não abra uma issue pública.

Envie um relato privado para:
- **E-mail**: `suporte@arayby.com`
- **Assunto**: `[YBYMO][SUPORTE] descrição curta`

Inclua, sempre que possível:
- Versão afetada;
- Passos para reproduzir;
- Impacto esperado/observado;
- Evidências (logs, payload, PoC);
- Sugestão de mitigação (se houver).

### Processo de tratamento

1. Confirmação de recebimento em até **3 dias úteis**.
2. Triagem inicial e classificação de severidade.
3. Atualizações periódicas de status a cada **7 dias corridos** até a conclusão.
4. Após correção, publicação da versão com patch e registro no `CHANGELOG.md`.

Relatos válidos recebem crédito na nota de release, salvo solicitação de anonimato.
